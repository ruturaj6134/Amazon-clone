# Amazon Clone

## Overview

This project is a responsive Amazon-inspired e-commerce frontend clone developed using HTML, CSS, and JavaScript. The objective of this project was to gain hands-on experience in front-end web development by recreating the layout and user interface of a modern e-commerce platform.

The application focuses on implementing responsive design principles, structured layouts, and an intuitive user interface while following best practices in web development.

## Features

- Responsive and modern user interface
- Amazon-inspired homepage layout
- Navigation bar with search functionality
- Product showcase sections
- Hero banner section
- Interactive hover effects
- Responsive design for desktop and mobile devices
- Organized and maintainable code structure

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Project Goals

The primary goals of this project were:

- Strengthen HTML, CSS, and JavaScript fundamentals
- Practice responsive web design techniques
- Improve understanding of layout systems such as Flexbox and CSS Grid
- Develop experience in building real-world user interfaces
- Learn project structuring and version control using Git and GitHub
